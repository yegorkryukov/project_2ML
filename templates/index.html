About PuddleNet

Our project will scrape 10 right-leaning and 10 left-leaning news sites to feed the content into a machine learning algorithm. We hope to teach the machine through unsupervised, deep learning, and natural language processing to determine if a body of text is right or left leaning.

Furthermore, as we create dimensions of data and extract features like date/time, title, text, authors, and keywords, we can begin to explore the data in new ways. For example, we can see if a topic tended to have the right and left meet more in the middle, or if it made them polar opposites. Viewing the same data, we can see if certain events led to bi-partisan alignment or if dates have an impact in general.
